single-page
===========

It's an example of a simple single page site generator.


Settings
--------
* settings.py


Input files
-----------
* src/index.css
* src/index.md
* templates/index.html


Output files
------------
* dist/index.css
* dist/index.html


Running
-------
    gena -s setting.py src


Version of GenA
---------------
    >=0.1
